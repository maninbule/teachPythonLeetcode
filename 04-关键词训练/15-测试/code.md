# (2) 给你两个长度一样的字符串s，t，将里面的字符进行交替，形成一个新的字符串，然后再返回
    例如：s: abc t:def  ->  adbecf

    例如：s: abcd t:xyzu  ->  abxycdzu

    例如："hello   world code  " -> "olleh   dlrow edoc  "
